# Changelog

## v1.0

Primera versión consolidada de ManipulAR.

### Características principales

- Entrenamiento RX.
- Entrenamiento TX.
- Decodificador.
- Test PARIS.
- Modo Keyer radio.
- Soporte para llave recta, iámbica A e iámbica B.
- Preferencias guardadas en EEPROM.
- Set Personal editable desde el equipo.
- Tono de audio configurable.
- Estructura modular del firmware.
- Copia local modificada de MorseEnDecoder con ajuste de tono en tiempo de ejecución.
