# Roadmap

Ideas y tareas pendientes para próximas versiones.

## Documentación

- Agregar esquemas de conexión.
- Agregar fotos del prototipo.
- Agregar capturas del equipo en funcionamiento.
- Ampliar el manual de usuario con ejemplos de sesiones de práctica.
- Documentar calibración o ajuste fino de velocidad si fuera necesario.

## Hardware

- Publicar un esquema recomendado para `KEY OUT` con optoacoplador.
- Documentar variantes de jack de audio externo.
- Revisar recomendaciones de gabinete y montaje.

## Firmware

- Probar el firmware en Arduino Uno, Nano y clones habituales.
- Revisar compatibilidad entre bibliotecas `LiquidCrystal_I2C`.
- Evaluar más mensajes de ayuda en pantalla sin complicar la interfaz.

## Comunidad

- Recibir feedback de usuarios que practiquen CW.
- Registrar problemas de montaje o uso frecuente.
- Convertir preguntas frecuentes en una wiki si el proyecto crece.
